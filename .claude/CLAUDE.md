# Claude Code Project Configuration

## Intelligent Closed Loop Workflow
This project follows a 6-stage intelligent development loop. Use `/dev-loop` to navigate it.

1.  **Plan**: `/feature-dev` - Architecture & Planning
2.  **Dev**: `/tdd` - Test-Driven Implementation
3.  **Verify**: `/verify` - CI/CD Quality Gates
4.  **Review**: `/code-review` - Multi-Agent Audit
5.  **Deploy**: `/commit-push-pr` - Git Automation
6.  **Learn**: `/learn` - Continuous Improvement

## Quick Start
*   **Start a new feature**: `/feature-dev`
*   **Check code quality**: `/code-review`
*   **Full workflow guide**: `/dev-loop`

## Tech Stack
- **Backend**: FastAPI + SQLModel + MySQL + LangGraph
- **Frontend**: Vue 3 + Element Plus + Vuex + TailwindCSS
- **AI/Agent**: Claude Code + MCP + LangChain
