# /check - 快速代码检查

## 描述
快速检查代码规范和常见问题。

## 使用方式
```
/check [目标] [--scope <范围>]
```

## 参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `目标` | 文件/目录路径 | 当前文件 |
| `--scope` | `file`/`changed`/`all` | `file` |

## 检查项

参考 `@templates/review-patterns.md` 中的审查清单：
- 代码规范（命名、长度、注释）
- 代码质量（异常处理、空值检查）
- 明显性能问题
- 明显安全问题

## 输出格式

```

║                    ✅ 代码检查通过                                ║
╠══════════════════════════════════════════════════════════════════╣
║  文件: 3    问题: 0                                              ║
╚══════════════════════════════════════════════════════════════════╝
```

或

```
╔══════════════════════════════════════════════════════════════════╗
║                    ⚠️ 发现 3 个问题                               ║
╠══════════════════════════════════════════════════════════════════╣
║  🟡 userService.py:45 - 函数过长 (65 行)                          ║
║  🟡 UserList.vue:10 - 缺少 key 属性                               ║
║  🔵 utils.py:20 - 建议添加类型注解                                ║
╚══════════════════════════════════════════════════════════════════╝
```

## 示例
```bash
/check                          # 检查当前文件
/check src/views/               # 检查目录
/check --scope changed          # 检查已修改文件
```
