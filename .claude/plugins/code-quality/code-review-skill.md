# Code Review Skill

代码审查技能，提供代码质量分析能力。

## 触发条件
- `/review` 命令
- `/check` 命令
- 用户请求代码审查

## 审查能力

### 代码规范检查
参考 `@templates/review-patterns.md`：
- 命名规范
- 代码重复
- 函数长度
- 注释完整性
- 类型注解

### 代码质量检查
- 异常处理
- 空值检查
- 边界条件
- 资源释放
- 死代码

### 性能问题识别
- N+1 查询
- 内存泄漏
- 阻塞操作
- 无效渲染

### 安全问题识别
- SQL 注入
- XSS
- 硬编码密钥
- 敏感日志

## 输出要求

1. 按严重级别分类（严重 > 警告 > 建议）
2. 提供具体文件和行号
3. 给出修复建议
4. 使用统一报告格式

## 与其他组件协作

- 深度安全检查 → 调用 `security` 插件
- 深度性能检查 → 调用 `debugger` 插件
- 需要上下文隔离 → 调用 `code-reviewer` Agent
