# 后端用例生成指南

本指南提供了后端测试用例生成的核心检测维度，帮助AI Agent生成全面、高质量的测试用例。

## 测试维度说明

### ✓ 安全漏洞检测 → 替代安全测试
- SQL注入
- 命令注入
- 认证鉴权
- 敏感信息泄露

### ✓ 功能性Bug → 替代功能测试
- 空值处理
- 异常捕获
- 事务回滚
- 并发安全

### ✓ 性能瓶颈 → 替代性能测试
- N+1查询
- 缺少索引
- 缓存策略
- 同步阻塞

### ✓ 数据库设计 → 替代数据测试
- 表结构
- 约束
- 字段类型
- 软删除

### ✓ 代码质量 → 替代代码审查
- 职责单一
- 代码重复
- 错误处理
- 日志记录

### ✓ API设计 → 替代接口测试
- RESTful规范
- 响应格式
- 接口文档
- 参数校验
