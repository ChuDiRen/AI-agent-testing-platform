# AI Agent Frontend

基于 Vue 3 + TypeScript + Vite 构建的AI智能代理前端应用。

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发环境启动
```bash
npm run dev
```
前端服务将运行在：http://localhost:5173

### 构建生产版本
```bash
npm run build
```

## 配置说明

- 前端开发服务器端口：5173
- 后端API服务端口：8000
- 开发环境下，前端会自动代理 `/api` 请求到后端服务

## 技术栈

- Vue 3 + TypeScript
- Element Plus UI组件库
- Vite 构建工具
- Axios HTTP客户端
- Vue Router 路由管理
- Pinia 状态管理
