This is a simple static HTML page that you can test with Playwright.

What you'll learn:

  • Loading local HTML files with the file:// protocol
  • Navigating to pages and waiting for them to load
  • Extracting page title and content
  • Understanding the Playwright sync API

Key Concepts:

Static HTML files don't require a web server. You can test them directly by converting the file path to a file:// URL.
