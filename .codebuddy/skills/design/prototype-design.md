# 原型设计 Skill

## 概述
本 Skill 用于根据需求文档生成高保真的原型界面，支持移动端和 PC 端应用。技术栈无关，使用纯 HTML + CSS 实现。

## 使用场景
- 快速出原型与老板/客户确认需求
- 为正式页面开发做准备
- UI/UX 设计验证

---

## 原型设计提示词模板

### 移动端 H5 原型
```prompt
你是一个专业的UI设计师，你需要根据我提供的需求文档来完成页面的设计

请仔细阅读需求文档 @[需求文档路径]，现在需要输出高保真的原型图，请通过以下方式帮我完成所有界面的原型设计，并确保这些原型界面可以直接用于开发：

1、用户体验分析：先分析这个 App 的主要功能和用户需求，确定核心交互逻辑。

2、产品界面规划：作为产品经理，定义关键界面，确保信息架构合理。

3、高保真 UI 设计：作为 UI 设计师，设计贴近真实 iOS/Android 设计规范的界面，使用现代化的 UI 元素，使其具有良好的视觉体验。

4、HTML 原型实现：使用 HTML + Tailwind CSS（或 Bootstrap）生成所有原型界面，并使用 FontAwesome（或其他开源 UI 组件）让界面更加精美、接近真实的 App 设计。拆分代码文件，保持结构清晰：

5、每个界面应作为独立的 HTML 文件存放，例如 home.html、profile.html、settings.html 等。

• index.html 作为主入口，不直接写入所有界面的 HTML 代码，而是使用 iframe 的方式嵌入这些 HTML 片段，并将所有页面直接平铺展示在 index 页面中，而不是跳转链接。

• 真实感增强：
  - 界面尺寸应模拟 iPhone 15 Pro，并让界面圆角化，使其更像真实的手机界面。
  - 使用真实的 UI 图片，而非占位符图片（可从 Unsplash、Pexels、Apple 官方 UI 资源中选择）。
  - 添加顶部状态栏（模拟 iOS 状态栏），并包含 App 导航栏（类似 iOS 底部 Tab Bar）。

请按照以上要求生成完整的 HTML 代码，并确保其可用于实际开发
```

### PC 端管理后台原型
```prompt
你是一个专业的UI设计师，你需要根据我提供的需求文档来完成管理后台页面的设计

请仔细阅读需求文档 @[需求文档路径]，现在需要输出高保真的管理后台原型图：

1、用户体验分析：分析管理后台的主要功能和用户需求，确定核心交互逻辑。

2、产品界面规划：定义关键界面，确保信息架构合理，包括：
   - 登录页面
   - 主控制台/仪表盘
   - 列表页面（带搜索、筛选、分页）
   - 详情页面
   - 表单页面（新增/编辑）

3、高保真 UI 设计：设计现代化的管理后台界面，包含：
   - 左侧导航菜单
   - 顶部工具栏
   - 面包屑导航
   - 数据表格
   - 统计卡片
   - 图表组件

4、HTML 原型实现：使用 HTML + Tailwind CSS 生成所有原型界面：
   - 每个页面作为独立的 HTML 文件
   - index.html 作为主入口，使用 iframe 平铺展示所有页面
   - 使用 Chart.js 或类似库展示图表

请按照以上要求生成完整的 HTML 代码
```

## 原型文件结构
```
prototype/
├── index.html          # 主入口，平铺展示所有页面
├── home.html           # 首页
├── list.html           # 列表页
├── detail.html         # 详情页
├── profile.html        # 个人中心
├── settings.html       # 设置页
├── css/
│   └── styles.css      # 自定义样式
└── js/
    └── main.js         # 交互逻辑
```

|-----|-----|

| 状态栏 | 模拟 iOS 状态栏 |

| 侧边栏宽度 | 240px |

## 工作流程

1. **阅读需求文档**：理解核心功能和用户需求
2. **规划页面结构**：列出所有需要的页面
3. **设计核心页面**：先完成主要功能页面
4. **完善细节**：添加交互效果和动画
5. **局部调整**：根据反馈进行修改
6. **确认交付**：与老板/客户确认需求
